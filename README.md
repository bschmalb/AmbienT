# AmbienT
Eine Prototype-Applikation, welche Hintergrundgeräusche für die Entspannung abspielt.

Studentisches Projekt welches im Rahmen der WBA1 Challenge 2017 der TH Köln am Campus Gummersbach entstanden ist.
